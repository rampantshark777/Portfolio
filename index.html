CS1300 Portfolio

Portfolio Introduction

In each assigned project, I aimed to pick a topic that I either interact with on a regular basis, have a strong passion for, or both. I wanted this to be the case in order for the work to feel meaningful. Furthermore, improving an interface that I know to have a real-world effect on people makes the process worthwhile.

While each project focuses on a different aspect of UI/UX, the overall objective to improve the user's experience by prioritizing simplicity and efficiency remains the same. It is this emphasis on thoughtfulness and empathy that I feel makes UI/UX special and attempted to embody in order to study and improve interfaces that I find meaningful.

Why this Interface?

Background

Creating an interface that is useful to users is paramount in UI/UX. In this project, I created a way for users to aggregate statistics about certain soccer players (specifically age and goals scored). The idea in this project is to be able to save the user time by making calculations behind the scenes and simply presenting a final product in a clear, understandable manner.

Why this Interface?

My decision to aggregate soccer statistics was influenced by what I felt I would find useful. As a passionate fan of the game, creating a page that would provide me with insights about players is something I find meaningful; while this project represent just a start given the time constraint, the potential for what an expanded version of this interface could be is visible.
